# StampTourApp

Jetpack Compose로 제작 중인 스탬프 투어 앱.  
하단 네비게이션을 기반으로 홈 / 지도 / 스탬프 / 마이페이지 구조를 구성함

## 🛠 Tech Stack
- Kotlin
- Jetpack Compose (Material3)
- Navigation Compose

## ✨ Features
- 홈 화면
  - 메인 배너 자동 슬라이드
  - 프로그램 / 이벤트 카드 슬라이더
  - 스탬프 진행률 표시
- 하단 네비게이션
  - 홈 / 지도 / 스탬프 / 마이페이지
- 화면 구조 분리(feature 기반)
